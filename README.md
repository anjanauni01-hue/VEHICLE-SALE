# Mawbima Motors Website

This is the premium vehicle sale website for Mawbima Motors.

## How to Run
It's very simple! Just:
1. Open the `VEHICLE-SALE` folder on your desktop.
2. Double-click on `index.html`.
3. The website will open in your browser immediately.

## Features
- **Supiri Design**: Modern, glassmorphism style with dark mode.
- **Mobile Responsive**: Works on phones and computers.
- **No Installation**: Uses Tailwind via CDN (internet connection required for styles).
- **Multiple Pages**: Home, Inventory, About, Contact.

## Editing
You can edit the HTML files directly to change text or images.
- Images are currently placeholders from Unsplash. You can replace the `src="..."` in the `<img>` tags with your own image files.
